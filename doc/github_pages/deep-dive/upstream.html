Platform: POSIX (Linux / WSL)

Simulation platform requiring no hardware. Runs the full lifecycle, CAN (via SocketCAN), Ethernet (lwIP), UDS, and all application-level systems on a desktop Linux or WSL environment. Primary platform for development and CI.

Features
  • Full lifecycle manager (9 run levels)
  • CAN via SocketCAN
  • Ethernet / lwIP / DoIP
  • UDS / DoCAN diagnostics
  • Console subsystem (stdio)
  • Logger, Tracing
  • Storage subsystem
  • FreeRTOS POSIX port
Testing
  • 1470 unit tests passing
  • LCOV code coverage
  • Puncover size analysis
  • CI: clang-tidy, clang-format

Platform: NXP S32K148EVB

Primary hardware platform. ARM Cortex-M4F @ 112 MHz, FlexCAN (CAN FD), Ethernet (ENET + TJA1101 PHY), FTM PWM, ADC, FlexNVM EEPROM. Full safety support (MPU, ECC, watchdog).
